AI Bias Detection with XAI

Fairness Analysis of Machine Learning Models using Explainable AI in R

📌 Project Overview

This project investigates bias in machine learning models using the Adult Income dataset. A Random Forest classifier is trained to predict income, and fairness is evaluated across demographic groups such as gender and race.

Explainable AI (XAI) techniques, including SHAP values and partial dependence plots, are used to understand model decisions and improve transparency.

🔗 GitHub Repository

✨ Features

  • Bias detection across gender and race
  • Random Forest classification model
  • SHAP explanations for individual predictions
  • Feature importance analysis
  • Partial dependence plots (global explainability)
  • Automatic saving of results (CSV, PNG, TXT)

⚙️ Tech Stack

  • R programming language
  • randomForest
  • iml (Explainable AI)
  • dplyr, ggplot2

📊 Results

  • Bias metrics showing differences in prediction rates
  • Visualization of fairness across groups
  • Identification of most influential features
  • Transparent explanations using SHAP

▶️ How to Run

Run the R script after installing dependencies:

install.packages(c("randomForest", "caret", "iml", "dplyr", "ggplot2"))

source("scripts/bias_analysis.R")
