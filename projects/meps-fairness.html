Algorithmic Fairness & Constrained AI Optimization

Auditing Structural Disparities and Mitigating Allocative Harms in Healthcare Access using Fairlearn

📌 Project Overview

This project investigates how systemic socioeconomic inequalities encode themselves into machine learning models within healthcare utilization pipelines. Using a survey-weighted microdata extract from the IPUMS Medical Expenditure Panel Survey (MEPS), a predictive system is audited to evaluate allocative harms concerning low-income populations.

By moving beyond "fairness through unawareness", the workflow implements active multi-model optimization using the Fairlearn framework to enforce mathematical parity constraints and navigate complex socio-technical trade-offs.

🔗 GitHub Repository

✨ Features

  • Pre-modeling disparity mapping via survey-weighted Nightingale Rose Charts [137]
  • Evaluation of "Fairwashing" mechanics within standard unaware classifiers [137]
  • Sample re-weighting audits to expose true baseline error differentials [137]
  • Post-processing optimization assessment via ThresholdOptimizer [137]
  • In-processing grid search optimizations under Demographic Parity and Equalized Odds boundaries [137]
  • Multi-objective performance mapping across the fairness-accuracy Pareto frontier [137]

⚙️ Tech Stack

  • Python / Jupyter Notebook [137]
  • Fairlearn (Algorithmic Mitigation Framework) [137]
  • Scikit-learn (Machine Learning Engine) [137]
  • Pandas, NumPy, Matplotlib, Jupytext [137]

📊 Results

  • Exposed a severe ~98% False Negative Rate (FNR) disparity gap during initial baseline audits [137].
  • Demonstrated the failure modes of naive class re-weighting and post-processing on uncalibrated healthcare features [137].
  • Achieved optimal deployment balance via Equalized Odds GridSearch, shrinking the low-income FNR to 31.1% while preserving predictive utility [137].

Clone the repository and run the setup using pip:

git clone https://github.com
cd meps-fairness-fairlearn

pip install fairlearn scikit-learn pandas numpy matplotlib requests jupytext

# (Optional) Regenerate the Jupyter notebook with local outputs
jupytext --to notebook plot_meps_healthcare_delay.py

⚖️ Ethical Considerations

When statistical estimators misclassify economically vulnerable individuals as "not at risk of delaying care" due to a lower baseline of interaction metrics, they codify allocative harms. This project highlights why sensitive socioeconomic domains require:

  • Rigorous integration of national survey sampling weights (PERWEIGHT) to preserve scientific validity [137].
  • Active mitigation algorithms rather than superficial statistical blindness [137].
  • Multi-objective optimization to guarantee equitable resource distribution [137].
