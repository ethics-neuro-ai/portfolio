EEGNet Epoch-Level – Sleep Deprivation EEG

EEGNet model predicting sleep deprivation states from resting-state EEG • ~96% accuracy

📌 Project Overview

This project uses the Sleep Deprivation EEG dataset to classify resting-state EEG epochs. The model is trained at the epoch level, aggregating EEG data from all subjects. It can distinguish EEG patterns between normal sleep and sleep-deprived sessions with high accuracy (~96%).

🔗 GitHub Repository ⬇️ Download Model 📓 Training Notebook

🧠 Dataset

The dataset includes:

  • 71 participants
  • Resting-state EEG (eyes open/closed)
  • Two sessions: Normal Sleep (NS) and Sleep Deprivation (SD)
  • Behavioral measures of sleepiness and mood
  • EEG preprocessed into epochs (~2s, 61 channels)

⚙️ Technologies

  • Python 3.x
  • TensorFlow & Keras
  • MNE-Python
  • NumPy & Pandas
